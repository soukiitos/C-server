Serveur Web HTTP simple en langage C

Ce programme implémente un serveur HTTP minimal qui écoute sur le port 8080.
Il accepte les connexions TCP entrantes, analyse les requêtes HTTP GET de base,et sert les fichiers HTML statiques du répertoire de travail actuel.Si un fichier demandé est introuvable, le serveur répond par une erreur 404.
Le serveur tourne en boucle infinie et peut être arrêté manuellement avec Ctrl+C et il eqt Compatible avec toutes les systèmes.
