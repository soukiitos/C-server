Bienvenue sur mon serveur web en langage C!

Cette page Web est servie à partir d’un serveur HTTP minimal écrit entièrement en C. Il est rapide, léger et démontre des compétences réseau de bas niveau.

Alors, Ce programme implémente un serveur HTTP minimal qui écoute sur le port 8080.
Il accepte les connexions TCP entrantes, analyse les requêtes HTTP GET de base,et sert les fichiers HTML statiques du répertoire de travail actuel.Si un fichier demandé est introuvable, le serveur répond par une erreur 404.

Le serveur tourne en boucle infinie et peut être arrêté manuellement avec Ctrl+C et il est Compatible avec toutes les systèmes.

pour travailler sur ce projet j'ai fait quelques étape pour réusir et obtient ce projet que j'ai de mon coeur, premiérement j'ai commencé par créer un fichier C (comme server.c) qui utilise des sockets pour écouter les requêtes HTTP. Ensuite, dans le terminal Ubuntu, je compile-le avec gcc server.c -o server. puis j'ai crée un fichier index.html dans le même dossier que le contenu de mon projet. Ensuite, j'éxecute ./server et on ouvre le navigateur à l'adresse http://localhost:8080 pour voir ce site en ligne. Bah, Cette configuration nous aide à comprendre les bases du fonctionnement des serveurs et de HTTP !

Vous pouvez explorer le code source du serveur ou personnaliser cette page en index.html.
